####Deskripsi Program #####
Program ini bertujuan untuk mencari Penyelesaian dan langkah-langkah dari Puzzle-15

####Requirement Program####
Java Runtime Environment

###Cara Menggunakan Program####
###Menjalankan Program####
1. Pastikan Java Runtime Environment sudah terinstal
2. Buka folder bin, buka CLI di folder tersebut
3. Jalankan command "java Main"
4. Program akan berjalan
5. Pilih apakah puzzle akan dibuat secara acak atau dari file text
6. Jika dari file text pastikan ubin yang kosong telah diganti dengan angka 16
7. Jika memilih file maka akan keluar gui untuk memilih file yang berisi Puzzle
8. JIka memilih acak maka program akan membuat puzzle secara acak
9. Program akan menghitung goal dan mencoba menyelesaikan puzzle jika goal bernilai genap
10. Jika simpul yang dibangkitkan cukup besar maka program akan menghentikan pencarian langkah untuk mencegah terjadinya error


####Kompilasi Program####
1. Pastikan Java Runtime Environment sudah terinstal
2. Buka folder src, buka CLI di folder tersebut
3. Jalankan command "javac puzzle.java Solve.java Main.java"
4. Jika kompilasi berhasil maka akan mengeluarkan file Main.class puzzle.class SimpanPilihFile.class dan Solve.class
5. Buka program sesuai dengan perintah "Menjalankan Program"
